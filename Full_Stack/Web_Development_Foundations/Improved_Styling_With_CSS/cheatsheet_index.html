Semantic HTML Reference

Semantic HTML Tags

Tag Name Description
<header> Header Describes the content at the top of the page.
<nav> Navigation Encapsulates the page's navigational links.
<main> Main Encapsulates the main content of a page between the header and footer areas.
<footer> Footer Includes the page's footer content at the bottom of the page.
<video> Video Allows the addition of videos to a website.
<audio> Audio Allows the implementation of audio into a website.
<figure> Figure Encapsulates media such as an image, diagram, or code snippet.
<figcaption> Figure Caption Describes the media encapsulated within the <figure> element.
<section> Section Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article Holds content that makes sense on its own such as articles, blogs, and comments.
<aside> Aside Marks additional information that can enhance another element but isn't required in order to understand the main content.
